# HEARTBEAT.md

Use this file for periodic cleanup and review prompts.

## At Session Start

- Check for recent notes worth carrying forward.
- Review `MEMORY.md` and any memory files if they exist.
- Notice anything that looks stale, repetitive, or out of sync with the app.
- If the UI or workflow changed, update the memory notes too.

## Periodic Review

- Merge duplicate notes.
- Promote stable facts into long-term memory.
- Remove or archive obsolete context when it stops being useful.
- Re-check whether active guidance still matches the current app shape.
