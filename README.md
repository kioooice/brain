# 💡 灵感收集工具

本地Web应用，用于收集和管理灵感。

## 功能

- ✅ 记录文字灵感
- ✅ 收集链接（GitHub、Twitter等）
- ✅ 上传图片/视频
- ✅ 分类和标签
- ✅ 状态标记（待整理/待实现/已实现/已归档）
- ✅ 搜索和筛选

## 快速启动

```bash
# Windows
双击 start.bat

# 或手动运行
pip install -r requirements.txt
python app.py
```

访问 http://localhost:5001

## 内容类型

| 类型 | 说明 |
|------|------|
| 📝 文字 | 文字灵感、想法 |
| 🔗 链接 | 网站、项目链接 |
| 🖼️ 图片 | 上传图片 |
| 🎬 视频 | 上传视频 |

## 技术栈

- Python Flask
- SQLite
- 原生HTML/CSS/JS
